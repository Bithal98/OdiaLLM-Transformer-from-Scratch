# OdiaLLM – Transformer from Scratch.

A 90M parameter Encoder-Decoder Transformer trained from scratch for Odia-English translation.
<img width="1625" height="1131" alt="output2" src="https://github.com/user-attachments/assets/336624a4-1d91-44c3-bf1d-6923ebc170a7" />

## 🚀 Features
- Built using "Attention Is All You Need"
- Trained on ~500K parallel sentences
- No pretrained models used
- Optimized for low-resource language (Odia)

## 📊 Results
- chrF Score: 36.45

- Python
- PyTorch
